# Aslan's Dotfiles

## Installation
1. `cd ~ && mkdir personal && cd personal`
2. clone the repo into `~/personal/dotfiles/` folder
3. `cd ~/personal/dotfiles`
4. Create your local git configuration: `cp .gitconfig.local.template .gitconfig.local` and edit with your personal info (name, email, GPG key)
5. Run setup: `./setup.sh`
6. install nerdfont https://github.com/ryanoasis/nerd-fonts/releases/download/v2.3.3/Hack.zip
7. install tmux plugins by `tmux` and hitting <ctrl><space> + I
